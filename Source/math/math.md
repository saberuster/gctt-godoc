<h2 id="Atan2">func <a href="//github.com/golang/go/blob/2ea7d3461bb41d0ae12b56ee52d43314bcdb97f9/src/math/atan2.go#L19">Atan2</a>
    <a href="#Atan2">¶</a></h2>
<pre>func Atan2(y, x <a href="/builtin/#float64">float64</a>) <a href="/builtin/#float64">float64</a></pre>

Atan2 returns the arc tangent of y/x, using the signs of the two to determine
the quadrant of the return value.

Special cases are (in order):

    Atan2(y, NaN) = NaN
    Atan2(NaN, x) = NaN
    Atan2(+0, x>=0) = +0
    Atan2(-0, x>=0) = -0
    Atan2(+0, x<=-0) = +Pi
    Atan2(-0, x<=-0) = -Pi
    Atan2(y>0, 0) = +Pi/2
    Atan2(y<0, 0) = -Pi/2
    Atan2(+Inf, +Inf) = +Pi/4
    Atan2(-Inf, +Inf) = -Pi/4
    Atan2(+Inf, -Inf) = 3Pi/4
    Atan2(-Inf, -Inf) = -3Pi/4
    Atan2(y, +Inf) = 0
    Atan2(y>0, -Inf) = +Pi
    Atan2(y<0, -Inf) = -Pi
    Atan2(+Inf, x) = +Pi/2
    Atan2(-Inf, x) = -Pi/2

<a id="exampleAtan2"></a>
Example:

    fmt.Printf("%.2f", math.Atan2(0, 0))
    // Output: 0.00
